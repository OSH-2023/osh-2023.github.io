## 本学期的实验安排

本学期的实验设计和之前几届的大致相同，但会结合前几届的经验和反馈进行调整和优化。实验的目的是让大家加深对操作系统诸概念和技术的理解。请大家务必认真完成实验，这是大家掌握操作系统重难点的绝佳机会。我们会在每个实验文档中尽可能给到精炼而有用的引导，帮助大家顺利完成实验。
